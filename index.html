Experience

Deputy Manager - Java Backend Developer

April 2024 – Present

Jio Platforms Limited

As a Deputy Manager at Jio Platforms Limited, I have contributed extensively to core backend development, telecom automation, and system integration across major projects like NEF (Network Exposure Function) and SON (Self-Organizing Network). I developed and maintained microservices using Core Java and Java 8, led the design and implementation of RESTful APIs, and integrated them with Redis for caching and session handling. In the NEF project, I developed a microservice called Analytics Exposure, handling end-to-end CRUD operations, testing via Postman, and ensuring high performance using Docker deployments. I also built automated upgrade and rollback mechanisms using shell scripts, monitored performance via JConsole, and continuously improved code quality using SonarQube.

In the SON project, I focused on developing 5G network automation modules to enhance performance and reduce manual intervention. This included implementing intelligent features for automated Neighbour Addition/Removal (NR & EUTRA), building advanced CLI commands for tasks such as cell locking/unlocking and GPS data retrieval, and enabling PAN India software upgrades through a single interface. I closely collaborated with cross-functional teams, integrated REST APIs with CLI tools to enable real-time data interaction with retry/error handling, and used Kibana for data visualization. These efforts not only streamlined operations but also improved scalability, reliability, and observability across large-scale telecom environments.
